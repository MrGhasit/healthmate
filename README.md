# HealthMate — Инструкция по запуску и сборке APK

## 📁 Структура проекта

```
healthmate/
├── main.py                    ← Весь Python-код
├── healthmate.kv              ← Дизайн и разметка
├── buildozer.spec             ← Конфигурация для сборки APK
├── .github/
│   └── workflows/
│       └── build.yml          ← GitHub Actions (авто-сборка)
└── images/
    ├── background.png         ← Фон (тёмный, 400x700)
    ├── logo.png               ← Логотип (100x100)
    └── avatar_default.png     ← Аватар (100x100)
```

---

## 🖥️ Запуск на компьютере (тест)

### 1. Установи Python 3.11 и зависимости:
```bash
pip install kivy==2.3.0
```

### 2. Запусти:
```bash
python main.py
```

---

## 📱 Сборка APK через GitHub Actions

### Шаг 1 — Создай репозиторий на GitHub
1. Зайди на https://github.com → New repository
2. Назови `healthmate`, сделай Public

### Шаг 2 — Загрузи файлы
```bash
git init
git add .
git commit -m "HealthMate v1.0"
git branch -M main
git remote add origin https://github.com/ТВО_ИМЯПОЛЬЗОВАТЕЛЯ/healthmate.git
git push -u origin main
```

### Шаг 3 — Дождись сборки
1. Зайди в репозиторий → вкладка **Actions**
2. Увидишь строку "Build HealthMate APK" — нажми
3. Жди 15–25 минут (первый раз долго — скачивает SDK)

### Шаг 4 — Скачай APK
1. После зелёной галочки → нажми на задачу
2. Внизу → раздел **Artifacts**
3. Скачай `HealthMate-debug-apk.zip`
4. Распакуй — внутри `healthmate-debug.apk`

---

## 📲 Установка на телефон
1. Перенеси APK на телефон (Telegram себе, USB, Google Drive)
2. На телефоне: **Настройки → Безопасность → Разрешить из неизвестных источников**
3. Нажми на APK → Установить

---

## 🖼️ Нужные картинки (создай сам или замени заглушками)

Создай папку `images/` и добавь туда любые PNG файлы с этими именами:
- `background.png` — тёмный фон 400×700
- `logo.png` — логотип 100×100
- `avatar_default.png` — аватар 100×100

Если картинок нет — приложение всё равно запустится (покажет пустые места).
